### 万字文档+健身房及会员小程序+Uniapp+SSM+Vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、微信开发者工具/HBuilderX
技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML、JavaScript、CSS、图片上传

系统功能模块：
管理员与用户2角色。
管理员：个人中心、用户管理、健身类型管理、健身教练管理、健身课程管理、健身预约管理、系统管理
用户：首页、健身教练、健身课程、我的信息、预约课程、健身课程、我的收藏管理
